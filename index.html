Surveillez LeBonCoin en temps réel

Finis les refresh manuels ! Vous recevez directement dans Telegram toutes les nouvelles annonces correspondant à vos filtres (prix, catégorie, zone géographique).

Fonctionnalités clés :

  • Alertes instantanées sur chaque nouvelle annonce
  • Filtres avancés par mot-clé et localisation
  • Tableau de bord web pour historiser et trier

Tech stack :

Python (BeautifulSoup, Requests), Flask, Telegram Bot API, MongoDB

Promotions en message privé

Un userbot Telegram 100 % sur mesure qui récupère tous vos flux promo (RSS, API e-commerce) et vous envoie les meilleures offres en DM, sans spam.

Fonctionnalités clés :

  • Envoi programmé selon vos plages horaires
  • Ségrégation par catégorie de vente
  • Commandes "/pause" et "/reprendre" intégrées

Tech stack :

Python (Aiogram), PostgreSQL, cron, React pour interface

Fonctionnalités clés :

  • Connexion automatique à +50 sources de données
  • Rapports PDF générés automatiquement
  • Alertes intelligentes sur les KPI critiques

Tech stack :

React, Node.js, PostgreSQL, Chart.js, API REST
